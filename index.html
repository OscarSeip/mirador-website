Fully Featured IIIF Viewer

Mirador is a configurable, extensible, and easy-to-integrate image viewer, which enables image annotation and comparison of images from repositories dispersed around the world. Mirador has been optimized to display resources from repositories that support the International Image Interoperability Framework (IIIF) API's. Mirador provides several workspaces for comparing image-based resources, suitable for use in both cultural heritage and research settings.

Classic Image Viewer Features

  • compare images
  • zoom / pan
  • scroll / book / gallery view

Rich Metadata display

  • metadata display: item and canvas level metadata
  • content navigation (table of contents)

Flexible Configuration

  • Multiple workspace types: Mosaic and Elastic
  • Workspace state: easily import and export your workspace

Annotations

  • annotation creation through a plugin
  • annotation viewing for OpenAnnotation and W3C Web Annotation

IIIF API Support

  • Supports IIIF v2 and v3 Image and Presentation APIs
  • IIIF Image Level 0 support
  • IIIF Authentication for limited-access resources
  • IIIF Search API for search-within

Multilingual Support

  • translated into 9 languages
  • easy to contribute new translations
  • Comprehensive right-to-left support: Mirador implements IIIF viewingDirection and HTML right-to-left can be set via config

Customizable Out of the Box

  • Rich set of out-of-the-box configuration
  • Themable (your organizations's branding!)
  • Light / Dark mode available

Extensible

  • Available as a single JS file or customizable build
  • And if you want more... the plugin framework enables extensive customization

Open Source, Community-Driven Software

  • built with an accessibility focus
  • friendly developer community
  • continuous integration and testing

Participate in the Community

Mirador's development, design, and maintenance is driven by community needs and ongoing feedback and discussion. Join us at our regularly scheduled community calls, on IIIF slack, or the mirador-tech and iiif-discuss mailing lists. To suggest features, report bugs, and clarify usage, submit a GitHub issue.
